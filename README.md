# OriEngine
Game engine made with Lightweight Java Game Library and OpenGL 3.2

Directory strucutre:
```
└───OriEngine
    ├───bin
    │   ├───engineTester
    │   └───renderEngine
    ├───lib
    │   ├───jars
    │   └───natives
    └───src
        ├───engineTester
        └───renderEngine
```
OriEngine/src/ <-- Contains the interesting files :)
OriEngine/lib/ <-- Contains [LWJGL2](https://github.com/LWJGL/lwjgl3) jars and natives
